# 📊 Data Science Job Analysis

## 📌 Problem Statement
Analyze global data science job trends to identify high-paying roles, in-demand skills, and job distribution.

## 🛠 Tools Used
- Excel
- Data Analysis
- Dashboarding

## 📂 Dataset
Public dataset containing job roles, salaries, skills, and locations.

## 🔍 Key Insights
- Data Scientist roles offer higher salary ranges compared to other roles
- Python and SQL are the most in-demand skills
- Job opportunities are highest in countries like USA and Australia

## 📊 Dashboard
Includes visualizations for:
- Salary distribution
- Job roles
- Skills demand
- Platform distribution

## 📎 Files Included
- Raw data
- Cleaned analysis file
- Dashboard file

## 🚀 Conclusion
This project helps understand the global demand and trends in data science jobs.
